Error Falta De Codigo

Este error ocurre cuando falta algo en el código.

Por lo general, esto se debe a que falta un punto y coma o un paréntesis de cierre.

El error ocurría en un código similar al siguiente:

if (solidom.equalsIgnoreCase("esfera"){
     vol=(4.0/3)*Math.pi*Math.pow(raiom,3);
}
else {
     if (solidom.equalsIgnoreCase("cilindro") {
          vol=Math.pi*Math.pow(raiom,2)*alturam;
     }
     else {
          vol=(1.0/3)*Math.pi*Math.pow(raiom,2)*alturam;
     }
}
return vol;

Por lo general, este mensaje de error no indica la ubicación exacta del problema. Para identificar el problema, debe:
     -Asegúrese de que todos los corchetes izquierdos tengan los corchetes derechos correspondientes.
     -Mire el código antes de la línea indicada por el error. Este error generalmente lo encuentra el compilador en el código que se encuentra detrás.
     -A veces, algunos caracteres (como el corchete izquierdo) no deben ubicarse en el primer código Java.
