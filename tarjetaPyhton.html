Errores de sintaxis

Estos errores de sintaxis suelen suceder por dos principales razones:

Olvidar en un if, elif, else, for, while, class o def agregar al final de la declaración: ( )

El error ocurría en un código similar al siguiente:

If spam == 42
         Print(‘Hello’)

Y además en usar = en lugar de ==, esto porque = es el operador de asignación y == es igual a la operación de comparación

El error ocurría en un código similar al siguiente:

If spam = 42
          Print(‘Hello’)
